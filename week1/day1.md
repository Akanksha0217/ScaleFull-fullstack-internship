## 💡 Notes

- **HTML5** introduces semantic elements like `<header>`, `<footer>`, `<article>`, etc.
- **Block elements** occupy full width and start on a new line (e.g., `<div>`, `<p>`)
- **Inline elements** flow within text (e.g., `<span>`, `<a>`, `<img>`)
